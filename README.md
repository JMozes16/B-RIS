## Development

### How To Run App
* Step 1: Clone Repository
* Step 2: Download NodeJS
* Step 3: Run npm install
* Step 4: Run npm run start
* Step 5: Go to http://localhost:3000
* Happy Hacking

### How To Test App
* Step 1: Complete Steps above if first time
* Step 2: Run npm run test
